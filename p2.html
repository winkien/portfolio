02 Summer 2020

Minion Surgery

Digital + Graphics

Individual Project


Project Description

In the Multimedia Programming for Art and Design course, I had to create an interactive multimedia simulation that demonstrates a process in an environment, with user interactivities being the major focus by using Java. I created a surgery simulation called Minion Surgery. The concept of this simulation is to help the Minion to do heart transplantation for replacing the old heart. There will be some information listed at the top of the screen and users need to follow the steps to complete the simulation.

Game start screen of Minion Surgery. [Screenshot]

Design Process

After deciding to create a surgery simulation, I created a proposal to describe the concept, the main coding and the users' interaction in the simulation. After that, I started to create my own images by sketching the elements one by one in Adobe Photoshop. At last, I applied coding to place the images in the right position, allow users to interact with this simulation by using mouse clicking and dragging and add music and sound to make the simulation more realistic.

Figure 2: Ng, W.(Summer 2020). Every elements of the simulation are created on my own by using Adobe Photoshop. [Screenshot]

Challenge

In order to meet the technical requirements, such as Perlin noise, recursive function, animation and decorator pattern, the simulation took a bit of time to run. The technical requirements also slow down the game process. Since the objects in the simulation are a bit small, it is hard for users to drag the objects to the proper locations. In order to solve these problems, I reduced the complexity of the technical elements and changed some of the dragging actions into clicking actions.

Reflection

This project allows me to design and develop a simulation that focuses on interactive human activities and dynamic environmental variations by combining my own imagination and creativity and coding techniques. It also taught me how to manage my time by breaking this process into different tasks since this project was very time-consuming. Overall, I think I did a great job on this project since I got full marks for this project and received compliments from my friends and TA.
